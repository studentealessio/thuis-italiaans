Estructura básica

Passato prossimo = auxiliar (essere o avere) + participio pasado.

  • Regular -are → participio en -ato: parlare → parlato
  • Regular -ere → participio en -uto: credere → creduto
  • Regular -ire → participio en -ito: dormire → dormito

Ejemplo: Ieri ho parlato con Maria. (Ayer hablé con Maria.)

¿Essere o avere?

La parte más engañosa. Reglas prácticas:

  • Movimiento o cambio de estado → essere: andare, venire, arrivare, partire, entrare, uscire, nascere, morire, diventare.
  • Todo lo demás → avere: mangiare, parlare, comprare, leggere, dormire...
  • Reflexivos → siempre essere: mi sono lavato.

Con essere, el participio concuerda con el sujeto: Maria è andata. Marco è andato. I bambini sono andati.

Participios irregulares clave

  • essere → stato / avere → avuto / fare → fatto / dire → detto
  • vedere → visto / venire → venuto / leggere → letto / scrivere → scritto
  • prendere → preso / mettere → messo / aprire → aperto / chiudere → chiuso
