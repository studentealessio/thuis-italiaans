La structure

Passato prossimo = auxiliaire (essere/avere) + participe passé.

  • -are → -ato : parlare → parlato
  • -ere → -uto : credere → creduto
  • -ire → -ito : dormire → dormito

Ieri ho parlato con Maria. (Hier j'ai parlé avec Maria.)

Essere ou avere ?

  • Mouvement ou changement d'état → essere : andare, venire, arrivare, partire, entrare, uscire, nascere, morire, diventare.
  • Le reste → avere : mangiare, parlare, comprare, leggere...
  • Verbes pronominaux → toujours essere : mi sono lavato.

Avec essere, le participe s'accorde avec le sujet : Maria è andata, Marco è andato, i bambini sono andati.

Participes irréguliers clés

  • essere → stato / avere → avuto / fare → fatto / dire → detto
  • vedere → visto / venire → venuto / leggere → letto / scrivere → scritto
  • prendere → preso / mettere → messo / aprire → aperto / chiudere → chiuso
