Struktur

Passato prossimo = Hilfsverb (essere/avere) + Partizip Perfekt.

  • -are → -ato: parlare → parlato
  • -ere → -uto: credere → creduto
  • -ire → -ito: dormire → dormito

Ieri ho parlato con Maria. (Gestern habe ich mit Maria gesprochen.)

Essere oder avere?

  • Bewegung oder Zustandswechsel → essere: andare, venire, arrivare, partire, entrare, uscire, nascere, morire, diventare.
  • Alles andere → avere.
  • Reflexive Verben → immer essere: mi sono lavato.

Bei essere kongruiert das Partizip mit dem Subjekt: Maria è andata, Marco è andato, i bambini sono andati.

Wichtige unregelmäßige Partizipien

  • essere → stato / avere → avuto / fare → fatto / dire → detto
  • vedere → visto / venire → venuto / leggere → letto / scrivere → scritto
  • prendere → preso / mettere → messo / aprire → aperto / chiudere → chiuso
